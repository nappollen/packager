# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.4] - 2024-12-19

### Changed
- Enchanced the "-" compatibility in package IDs when creating packages
- Rename "{NAME}" template to "{REPOSITORY}" for clarity and create a thruly "{NAME}" template

## [0.0.3] - 2024-12-14

### Added
- GPG signing for exported packages
- Support "-" in package IDs

## [0.0.2] - 2024-12-13

### Added
- Search and filter in package list
- Package creator with template system

### Changed
- Improved UI layout and styling

### Fixed
- Package icon loading issues

## [0.0.1] - 2024-12-11

### Added
- Initial release
- Package browser window
- Package exporter functionality
- Gitignore parser for export filtering
- Support for VPM package format
